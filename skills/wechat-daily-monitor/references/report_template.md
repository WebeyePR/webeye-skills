# 公众号每日监控报告模板

生成报告时，按以下结构组织内容：

---

```markdown
# 公众号每日监控报告

> 监控时间：YYYY-MM-DD HH:MM
> 监控账号数：XX 个

---

## 更新概览

| 公众号 | 更新状态 | 文章数 | 最后更新时间 |
|--------|----------|--------|--------------|
| [公众号A](#公众号a) | ✓ 已更新 | 2篇 | HH:MM |
| [公众号B](#公众号b) | ✓ 已更新 | 1篇 | HH:MM |
| [公众号C](#公众号c) | ✗ 未更新 | 0篇 | - |
| [公众号D](#公众号d) | ⚠️ 获取失败 | - | - |

---

## 今日发文详情

### 公众号A（2篇）

1. **[头条] 文章标题**
   - 发布时间：HH:MM
   - 链接：https://mp.weixin.qq.com/...
   - 摘要：（如有）

2. **[二条] 文章标题**
   - 发布时间：HH:MM
   - 链接：https://mp.weixin.qq.com/...

---

### 公众号B（1篇）

1. **[头条] 文章标题**
   - 发布时间：HH:MM
   - 链接：https://mp.weixin.qq.com/...

---

### 公众号C

> 今日未更新

---

### 公众号D

> ⚠️ 获取失败：[错误原因]

---

## 统计摘要

| 指标 | 数值 |
|------|------|
| 监控账号总数 | XX 个 |
| 有更新账号 | XX 个 |
| 未更新账号 | XX 个 |
| 获取失败 | XX 个 |
| 今日发文总数 | XX 篇 |

### 时间分布

- 最早发文：HH:MM（公众号名）
- 最晚发文：HH:MM（公众号名）

### 发文排行

| 排名 | 公众号 | 文章数 |
|------|--------|--------|
| 1 | 公众号A | X篇 |
| 2 | 公众号B | X篇 |
| 3 | 公众号C | X篇 |

---

## 备注说明

- 部分账号可能因屏蔽搜索无法获取数据
- 群发文章有推送通知，发布文章无通知
- 数据来源：极致了 API

---

> 报告生成时间：YYYY-MM-DD HH:MM
```

---

## 报告生成要点

1. **状态标识**
   - ✓ 已更新：当天有发文
   - ✗ 未更新：当天无发文
   - ⚠️ 获取失败：API调用失败或账号异常

2. **文章位置**
   - 头条：position=1
   - 二条：position=2
   - 以此类推

3. **错误处理**
   - 账号不存在：提示"未找到该公众号"
   - 账号屏蔽：提示"该账号屏蔽搜索"
   - 网络错误：提示"网络错误，请稍后重试"
