# 错误码列表

## 错误码返回示例
调用 Open API 时，会通过 code 参数返回响应码和响应信息，如下所示。当响应码为 0 时，代表成功，否则为失败。各类型错误码的含义见下方列表：

错误码和响应信息示例：
```json
{
  "code": "k_op_ec_10100",
  "msg": "Request parameters are not valid: userId is empty"
}
```
