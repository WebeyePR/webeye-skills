# 查询用户记录列表

获取「用户」对象中符合指定条件的记录列表。Tips：可在 “目录>历史版本” 的接口文档中查看历史接口

## 请求方式
POST

## 接口路径
/v1/data/namespaces/ticket_management__c/objects/_user/records_query

## 请求

### 请求头
| 名称 | 类型 | 说明 | 示例值 |
| :--- | :--- | :--- | :--- |
| Authorization | 必填 string | 传入应用 Token | T:f582c06c6ed14686b0ef... |
| Content-Type | 必填 string | 类型为 application/json | application/json |

### 请求体
| 名称 | 类型 | 说明 | 示例值 |
| :--- | :--- | :--- | :--- |
| select | 必填 string[] | 需要获取的字段，使用字段唯一标识符进行查询，关联字段可使用「.」进行下钻 | ["_id", "_createdBy"] |
| filter | object | 筛选条件，通过 JSON 格式指定条件。 | -- |
| &nbsp;&nbsp;└─ conditions | object[] | 条件数组 | -- |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ operator | string | 操作符 | -- |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ left | object | 左值 | -- |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└─ type | string | 左值类型，只支持 "metadataVariable"，表示字段 | "metadataVariable" |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└─ settings | json | 字段具体值，格式：`{"fieldPath":[{"fieldApiName": "字段名","objectApiName": "对象名"}]}` | -- |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ right | object | 右值 | -- |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└─ type | string | 右值类型，只支持 "constant"，表示常量 | "constant" |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└─ settings | json | 常量具体值，格式：`{"data":"常量具体值"}` | -- |
| &nbsp;&nbsp;└─ expression | string | 描述 conditions 之间的逻辑关系，从 1 开始下标，支持 `and` 和 `or` | "1 and 2" |
| order_by | object[] | 排序参数，field 为字段，direction 为方向。 | -- |
| group_by | object[] | 聚合参数，field 为参与聚合的字段。 | -- |
| page_token | string | 分页 Token。第一页填空字符串 ""，不能与 OFFSET 一起使用。 | "" |
| use_page_token | bool | 是否使用 page_token 功能。默认为 False 。 | false |
| page_size | int | 期望返回条数，上限 500，默认 500。 | 100 |
| offset | int | 记录偏移量，默认 0。建议使用 ID 游标分页。 | 0 |
| need_total_count | bool | 是否返回记录总数（Total）。默认为 False。 | false |
| query_deleted_record | bool | 是否查询已删除记录。 | false |

## 响应
| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | string | 响应码，0 代表成功 |
| msg | string | 响应描述 |
| data | object | 符合条件的记录信息 |
| &nbsp;&nbsp;└─ next_page_token | string | 下一页的起始位置 Token |
| &nbsp;&nbsp;└─ total | int | 符合条件的记录数 |
| &nbsp;&nbsp;└─ items | object[] | 符合条件的记录列表 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _createdBy | object | 「创建人」，关联「用户」对象 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└─ _id | string | 关联用户对象的记录 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _id | string | 记录 ID |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _accountStatus | string | 「账号状态」选项：_used, _unused |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _name | object | 「姓名」，多语文本字段 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└─ zh_cn | string | 中文内容 |
| &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;└─ en_us | string | 英文内容 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _enableAccountPassword | bool | 「启用账号密码」 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _active | bool | 「激活状态」 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _updatedAt | int | 「更新时间」，Unix 时间戳 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _updatedBy | object | 「更新人」，关联「用户」对象 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _lark_user_id | string | 「飞书用户 ID」 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _createdAt | int | 「创建时间」，Unix 时间戳 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _avatar | object | 「头像」，头像 / 标识字段 |
| &nbsp;&nbsp;&nbsp;&nbsp;└─ _email | string | 「邮箱」 |
