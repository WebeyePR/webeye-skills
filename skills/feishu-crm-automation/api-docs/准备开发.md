# 准备开发

飞书 aPaaS 为每个应用自动生成了一系列 Open API。通过 Open API，你可以：读写应用内的记录，操作附件数据，以及执行应用内函数。

## 响应
调用 Open API 后的响应，将包含以下参数：

- **响应码**：调用该 API 返回的结果，包含在 code 中。0 为成功，其余为失败，详见「错误码列表」
- **响应描述**：调用 API 的响应描述，包含在 msg 中
- **返回数据**：调用 API 后返回的具体内容，包含在 data 中
- **记录总数**：当通过记录列表类型的接口获取数据时会额外返回记录的总数，以便查询完整列表

## 调用流程
调用 API 前，需要先获取应用 Token，应用 Token 的获取方式如下：

- **步骤 1**：由管理员在 API 凭证管理中新建。线上环境和开发环境的凭证不共享，需要在各自环境的应用管理中新建
- **步骤 2**：新建 API 凭证后，可获得 Client ID 和 Client Secret
- **步骤 3**：通过 Client ID 和 Client Secret 调用 「获取应用 Token」接口，即可获得 Token
- **步骤 4**：调用接口时，在请求头中的 Authorization 参数中携带 Token 即可操作授权范围内的数据

## 调用频率限制
每个 API 凭证的调用频率限制为 15 QPS，超过此限制时，会返回 code 为 k_op_ec_20003 的错误。
