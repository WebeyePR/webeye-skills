# 查询客户公司记录列表

获取「客户公司」对象中符合指定条件的记录列表。Tips：可在 “目录>历史版本” 的接口文档中查看历史接口

## 请求方式
POST

## 接口路径
/v1/data/namespaces/ticket_management__c/objects/object_customer_company/records_query

## 请求

### 请求头
| 名称 | 类型 | 说明 | 示例值 |
| :--- | :--- | :--- | :--- |
| Authorization | 必填 string | 传入应用 Token | T:f582c06c6ed14686b0ef.eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9... |
| Content-Type | 必填 string | 类型为 application/json | application/json |

### 请求体
| 名称 | 类型 | 说明 | 示例值 |
| :--- | :--- | :--- | :--- |
| select | 必填 string[] | 需要获取的字段，使用字段唯一标识符进行查询，关联字段可使用「.」进行下钻 | ["_id", "_createdBy"] |
| filter | object | 筛选条件 | -- |
|   └─ conditions | object[] | 条件数组 | -- |
|     └─ operator | string | 操作符 | "equals" |
|     └─ left | object | 左值 | -- |
|       └─ type | string | 左值类型，只支持 "metadataVariable" | "metadataVariable" |
|       └─ settings | json | 字段路径配置 | {"fieldPath":[{"fieldApiName": "字段名","objectApiName": "对象名"}]} |
|     └─ right | object | 右值 | -- |
|       └─ type | string | 右值类型，只支持 "constant" | "constant" |
|       └─ settings | json | 常量值配置 | {"data":"常量具体值"} |
|   └─ expression | string | 逻辑关系描述 | "1" |
| order_by | object[] | 排序参数 | -- |
| group_by | object[] | 聚合参数 | -- |
| page_token | string | 分页 Token | "" |
| use_page_token | bool | 是否使用 page_token 功能 | false |
| page_size | int | 期望返回的记录条数，上限 500 | 100 |
| offset | int | 返回记录的偏移量 | 0 |
| need_total_count | bool | 是否返回记录总数 | false |
| query_deleted_record | bool | 是否查询已删除记录 | false |

## 响应
| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | string | 响应码，0 代表成功 |
| msg | string | 响应描述 |
| data | object | 符合条件的记录信息 |
|   └─ next_page_token | string | 下一页 Token |
|   └─ total | int | 符合条件的记录数 |
|   └─ items | object[] | 记录列表 |
|     └─ _id | string | 记录 ID |
|     └─ _updatedAt | int | 更新时间（Unix 时间戳） |
|     └─ option_type | string | 服务等级（选项 API 名称） |
|     └─ text_name | string | 客户名称 |
|     └─ _createdBy/UpdatedBy | object | 创建人/更新人信息 |
|     └─ _name | object | 展示名称（多语） |
|     └─ _createdAt | int | 创建时间（Unix 时间戳） |
|     └─ option_status | string | 客户分级（选项 API 名称） |
|     └─ text_billing | string | 所属 Billing |

### Python 示例：
```python
import requests
import json

url = "https://ae-openapi.feishu.cn/v1/data/namespaces/ticket_management__c/objects/object_customer_company/records_query"

payload = json.dumps({
  "filter": {
    "conditions": [
      {
        "operator": "contains",
        "left": {
          "type": "metadataVariable",
          "settings": "{\"fieldPath\":[{\"fieldApiName\": \"text_name\",\"objectApiName\": \"object_customer_company\"}]}"
        },
        "right": {
          "type": "constant",
          "settings": "{\"data\":\"测试\"}"
        }
      }
    ],
    "expression": "1"
  },
  "select": [
    "_id",
    "text_name",
    "option_status"
  ],
  "page_size": 100
})

headers = {
  "Authorization": "YOUR_APP_TOKEN",
  "Content-Type": "application/json"
}

response = requests.request("POST", url, headers=headers, data=payload)
print(response.text)
```
