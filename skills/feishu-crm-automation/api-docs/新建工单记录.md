# 新建工单记录

在「工单」对象中新建记录。Tips：可在 “目录>历史版本” 的接口文档中查看历史接口

## 请求方式
POST

## 接口路径
/v1/data/namespaces/ticket_management__c/objects/object_order/records

## 请求

### 请求头
| 名称 | 类型 | 说明 | 示例值 |
| :--- | :--- | :--- | :--- |
| Authorization | 必填 string | 传入应用 Token | T:f582c06c6ed14686b0ef.eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9... |
| Content-Type | 必填 string | 类型为 application/json | application/json |

### 请求体
| 名称 | 类型 | 说明 | 示例值 |
| :--- | :--- | :--- | :--- |
| **record** | object | -- | -- |
| lookup_product | 必填 object | 「所属产品」，关联对象字段。 关联「产品」对象。 关联单条记录 | -- |
|   └─ _id | 必填 string | 关联记录的记录 ID | "100" |
| option_level | 必填 string | 「工单级别」，选项字段。 包含：P1 (option_blocked), P2 (option_major), P3 (option_minor), P4 (option_38de4642f70) | "option_blocked" |
| option_satisfaction | string | 「客户工单满意度」，选项字段。 包含：满意 (option_satisfied), 一般 (option_generally), 不满意 (option_dissatisfied) | "option_satisfied" |
| lookup_creator | object | 「工单发起人」，关联对象字段。 关联「用户」对象。 关联单条记录 | -- |
|   └─ _id | string | 关联用户对象的记录 ID | "100" |
| datetime_resolved | int | 「工单关闭时间」，日期时间字段。 使用 Unix 时间戳 | 1773653571329 |
| datetime_report | 必填 int | 「客户报告时间」，日期时间字段。 使用 Unix 时间戳 | 1773653571329 |
| lookup_company | 必填 object | 「客户名称」，关联对象字段。 关联「客户公司」对象。 关联单条记录 | -- |
|   └─ _id | 必填 string | 关联记录的记录 ID | "100" |
| lookup_handler | object | 「当前处理人」，关联对象字段。 关联「用户」对象。 关联单条记录 | -- |
|   └─ _id | string | 关联用户对象的记录 ID | "100" |
| autoid_no | string | 「工单编号」，自动编号字段。 为空时将根据规则自动生成编号 | "100" |
| option_status | string | 「处理状态」，选项字段。 包含：待处理 (option_todo), 处理中 (option_inprocess), 待验证 (option_tobeconfirmed), 已关闭 (option_resolved) | "option_todo" |
| option_type | 必填 string | 「工单类型」，选项字段。 包含：问题处理 (option_cosultation), 操作变更 (option_defects), Billing问题 (option_issues), 技术方案或报价 (option_complaint) | "option_cosultation" |
| attachment_file | object[] | 「工单附件」，文件字段 | -- |
|   └─ id | 必填 string | 文件的唯一标识 | "625d2f602af94d46972073db32a99ed2" |
|   └─ size | 必填 string | 文件大小，单位为 B | "1024" |
|   └─ mime_type | 必填 string | 文件的 MIME 类型 | jpg |
|   └─ name | 必填 string | 文件名称 | sample.jpg |
| datetime_proposed | int | 「工单发起时间」，日期时间字段。 使用 Unix 时间戳 | 1773653571329 |
| customer_reporter | string | 「客户提出人」，文本字段 | "Sample text" |
| text_title | 必填 string | 「工单标题」，文本字段 | "Sample text" |
| text_describe | string | 「工单描述」，文本字段 | "Sample text" |
| lookup_c5de0a7f9a4 | object[] | 「抄送」，关联对象字段。 关联「用户」对象。 关联多条记录 | -- |
|   └─ _id | string | 关联用户对象的记录 ID | "100" |
| text_6f3346e6b64 | string | 「GCP工单号」，文本字段 | "Sample text" |
| text_result | string | 「工单处理结果」，文本字段 | "Sample text" |

## 响应
| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| code | string | 响应码，0 代表成功，否则为失败，详情见错误码列表 |
| msg | string | 响应描述 |
| data | object | -- |
|   └─ id | int | 新建记录的记录 ID |
