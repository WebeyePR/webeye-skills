# Example Skill: Text Summarizer

这是一个示例技能，演示了如何在本库中组织代码和文档。

## 功能描述

该技能提供简单的文本摘要功能。目前作为一个占位示例，后续可以集成 OpenAI/Anthropic 等 API。

## 快速开始

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行示例：
   ```bash
   python scripts/summarizer.py
   ```

## 配置

在 `config.py` 中管理相关参数。
